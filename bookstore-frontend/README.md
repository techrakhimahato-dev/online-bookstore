# Patan Dhoka Books Inventory

A modern React + Tailwind CSS bookstore inventory management website with a customer catalog, book details, stock availability, and a secure demo admin dashboard.

## Run locally

```powershell
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://127.0.0.1:5173/
```

## Demo admin login

```text
Email: admin@patandhokabooks.com
Password: patan123
```

## Where to edit

- `src/main.jsx` - customer catalog, admin login, add/edit/delete/update stock logic, cover upload, and availability status
- `src/styles.css` - Tailwind imports and reusable component styles
- `index.html` - page shell and browser title
- `package.json` - npm scripts and dependencies

## Build check

```powershell
npm run build
```

This creates the production files in `dist/`.
